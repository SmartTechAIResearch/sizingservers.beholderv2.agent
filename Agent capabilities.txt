Baseboard
	Manufacturer, Model, Serial number (unique), BIOS
CPU
	Name
Disk
	Model, Serial number (unique)
Machine
	Hostname (unique), IPs (array), BMC IP, OS
NIC
	Name, MAC address (unique)
RAM
	Size (long, GB), Manufacturer, Part number, Serial number (unique), Type, Speed (long, Mhz)
	
Host
    All of the above if the machine is a vmware VM and the vmware tools are installed
